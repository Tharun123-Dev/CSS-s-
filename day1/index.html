Cascading style sheets

It is a styling language used to describe the look and formatting of a webpage in html. it can controls colors, fonts, layouts, spacing, animations and more.>

Purpose of CSS

1.it seperates content (HTML) from presentation (styles).

2.improves design connsistency.

3. makes websites responsive

Types of CSS

  1. Inline CSS
  2. Internal Css
  3. External CSS

Inline Css

written directly inside an Html element

Internal Css

Written inside a style tag within the Html file

External Css

written in a seperate .css file and linked to the html

Selectors

A selector used to target HTML elements and so you can apply styles to them

Types of selectors

  1. Universal Selector(*)
  2. Type selector
  3. Class Selector
  4. ID selector
  5. Group Selector
  6. Space Selector
  7. Child selector
  8. adjacent Sibling Selector
  9. Attribute Selector
  10. General Sibling selector
